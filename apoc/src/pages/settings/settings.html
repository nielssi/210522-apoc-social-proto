Settings App Preferences

Shake for random question

This option allows to automatically switch to a random question when the device is shaken.

 Push Notification Preferences

All updates

You receive all updates as push notifications, including: new problems, group activity and social interactions.

Group and Problem updates

You receive push notifications when group activity occurs or new problems are added.

Problem updates

You only receive push notifications when new problems are added.

No updates

You do not receive any push notifications from apoc.

 Privacy Preferences

Share your activity

Problems you have commented on, problems and comments you have rated, and any problems you have marked as solved will be shared in the global feed.

Don't share activity

This option will exclude your activity from the global feed, entirely.

 Cloud Preferences

Offline Availability for Problems

You can download all existing problem data while in wifi as to reduce loading times and data costs by activating this option.
